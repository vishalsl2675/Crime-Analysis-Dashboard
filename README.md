# Crime Analysis in India Dashboard
The Crime Analysis in India Dashboard is an interactive analytical solution designed to uncover patterns, trends, and insights related to crime incidents across India.
Built using Power BI, the dashboard transforms raw crime data into meaningful visual stories, helping stakeholders understand crime distribution, temporal trends, and resolution effectiveness.
________________________________________
## Objectives of the Dashboard
The dashboard provides clear insights into:
1.	Overall crime volume across India
2.	Year-wise crime trends
3.	City-wise crime distribution
4.	Crime occurrence by month, weekday, and time range
5.	Solved vs unsolved crime rate
6.	Seasonal and monthly crime patterns
